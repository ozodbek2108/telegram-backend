📣 E'lon yuborish
📬 Yuborilgan bildirishnomalar
📡
Bu yerda backendga qaysi Telegram akkaunti ulanishini siz o'zingiz, kodga tegmasdan, to'g'ridan-to'g'ri shu sahifadan sozlaysiz. API ID/Hash, telefon raqam, tasdiqlash kodi va (agar yoqilgan bo'lsa) 2 bosqichli himoya paroli — barchasi shu yerdan kiritiladi. Boshqa akkauntga o'tmoqchi bo'lsangiz, avval "Uzish" tugmasini bosing, so'ng yangi raqam bilan qaytadan ulang.
Yuklanmoqda…
📘 Qanday ulash kerak — to'liq qo'llanma
  1. 1️⃣API ID va API Hash oling. my.telegram.org saytiga öz telefon raqamingiz bilan kiring → "API development tools" bo'limiga o'ting → yangi ilova yarating (nomi va platforma ixtiyoriy) → sizga api_id (raqam) va api_hash (harf-raqamli kod) beriladi.
  2. 2️⃣Shu sahifaga API ID, API Hash va telefon raqamingizni kiriting (xalqaro formatda, masalan +998901234567) va "Kod yuborish" tugmasini bosing.
  3. 3️⃣Telegram ilovangizga kod keladi (Telegram'ning rasmiy "Telegram" xabari sifatida, ba'zan SMS orqali ham kelishi mumkin). Kelgan raqamlarni shu sahifadagi "Kod" maydoniga kiriting va "Tasdiqlash" tugmasini bosing.
  4. 4️⃣Agar akkauntingizda 2 bosqichli himoya (Two-Step Verification / Cloud Password) yoqilgan bo'lsa, tizim avtomatik ravishda qo'shimcha "Parol" maydonini chiqaradi — o'sha parolni kiriting va yana tasdiqlang. Agar 2 bosqichli himoya yoqilmagan bo'lsa, bu qadam umuman chiqmaydi.
  5. ✅Tayyor! Holat "✅ Ulangan" deb o'zgaradi va shu raqam ko'rsatiladi. Endi "Video yuklash" bo'limidan Telegram havolalari orqali video qo'sha olasiz.
  6. 🔁Boshqa akkauntga o'tish: "Uzish" tugmasini bosing (joriy sessiya butunlay o'chadi), so'ng yangi raqam bilan 1-qadamdan qaytadan boshlang.
⚠️ API Hash va tasdiqlash kodi — akkauntingizga to'liq kirish huquqi beradi. Bu sahifani faqat ishonchli adminlar ko'rishi kerak.
